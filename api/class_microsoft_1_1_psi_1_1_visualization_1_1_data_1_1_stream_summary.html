Detailed Description

Represents the summarized data of a stream generated from a specified summarizer type. Incoming data is summarized over a fixed interval and the resulting summarized values are stored in the cache as IntervalData items.

Template Parameters
TSrcThe type of stream messages.
TDestThe type of the summarized interval data.
