Detailed Description

Implements the splitter for the ParallelSparseDo<TIn, TBranchKey, TBranchIn> and ParallelSparseSelect<TIn, TBranchKey, TBranchIn, TBranchOut, TOut> components.

Template Parameters
TInThe input message type.
TBranchKeyThe key type.
TBranchInThe branch input message type.
TBranchOutThe branch output message type.
