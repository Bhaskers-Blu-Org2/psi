Detailed Description

This is the base class for any component that transforms an input type into an output type. Derive from thsi class if your component has more than one input or more than one output. Otherwise, use one of the the Operators.Select<TIn, TOut>(IProducer<TIn>, System.Func<TIn, Envelope, TOut>, DeliveryPolicy) or Operators.Process<TIn, TOut>(IProducer<TIn>, System.Action<TIn, Envelope, Emitter<TOut>>, DeliveryPolicy) operators.
