Detailed Description

Component that captures and streams audio from an input device such as a microphone.

This sensor component produces an audio output stream of type AudioBuffer which may be piped to downstream components for further processing and optionally saved to a data store. The audio input device from which to capture may be specified via the AudioCaptureConfiguration.DeviceName configuration parameter (e.g. "plughw:0,0").

This sensor component produces an audio output stream of type AudioBuffer which may be piped to downstream components for further processing and optionally saved to a data store. The audio input device from which to capture may be specified via the AudioCaptureConfiguration.DeviceName configuration parameter. The GetAvailableDevices static method may be used to enumerate the names of audio input devices currently available on the system.
Please note: This component uses Audio APIs that are available on Windows only.
