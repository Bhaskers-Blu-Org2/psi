Detailed Description

Component that performs body tracking from the depth/IR images captured by the Azure Kinect sensor.

It is important that Depth and IR images do not go through lossy encoding/decoding (e.g., JPEG) before arriving at the tracker. Unencoded or non-lossy (e.g. PNG) encoding are okay.
