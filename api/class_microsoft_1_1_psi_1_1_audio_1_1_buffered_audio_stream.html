Detailed Description

Provides a blocking, buffered audio stream using a circular buffer.

Constructor & Destructor Documentation

◆ BufferedAudioStream()

Microsoft.Psi.Audio.BufferedAudioStream.BufferedAudioStream ( long  capacity)

Initializes a new instance of the BufferedAudioStream class.

Parameters
capacityThe capacity of the internal buffer.

This class implements a blocking stream such that its Read and Write methods will not return until the specified number of bytes have been read or written. This is mainly to support streaming audio to and from .NET System.Speech classes that read from or write to System.IO.Stream using file stream semantics.
