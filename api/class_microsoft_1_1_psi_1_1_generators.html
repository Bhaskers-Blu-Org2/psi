Generates stream containing a single message, and keeps the stream open afterwards.

Template Parameters
TThe type of value to publish.
Parameters
pipelineThe pipeline to attach to.
valueThe value to publish.
intervalThe desired time interval used to align the messages with an alignment time. Defaults to 1 tick.
alignmentDateTimeIf non-null, this parameter specifies a time to align the generator messages with. If the paramater is non-null, the messages will have originating times that align with (i.e., are an integral number of intervals away from) the specified alignment time.
Returns
A stream of values of type T.

The generated stream stays open until the pipeline is shut down.

Parameters

countThe number of messages to publish.

Returns
A stream of consecutive integers.
Parameters
