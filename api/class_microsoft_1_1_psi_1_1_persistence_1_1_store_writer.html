Detailed Description

Implements a storage writer that can write multiple streams to the same file, while cataloging and indexing them.

Creates a logical storage stream to write messages to.

Parameters
streamIdThe id of the stream, unique for this store. All messages with this stream id will be written to this storage stream.
streamNameThe name of the stream. This name can be later used to open the sorage stream for reading.
indexedIndicates whether the stream is indexed or not. Indexed streams have a small index entry in the main data file and the actual message body in a large data file
typeNameA name identifying the type of the messages in this stream. This is usually a fully-qualified type name or a data contract name, but can be anything that the caller wants.
Returns
The complete metadata for the storage stream just created.
