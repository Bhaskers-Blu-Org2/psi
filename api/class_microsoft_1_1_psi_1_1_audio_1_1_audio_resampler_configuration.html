Detailed Description

Represents the configuration for the AudioResampler component.

Use this class to configure a new instance of the AudioResampler component. Refer to the properties in this class for more information on the various configuration options.

◆ InputFormat

WaveFormat Microsoft.Psi.Audio.AudioResamplerConfiguration.InputFormat
getset

Gets or sets the input format of the audio stream to be resampled.

This specifies the expected format of the audio arriving on the input stream. If not set, the AudioResampler component will attempt to infer the audio format from the AudioBuffer messages arriving on the input stream.

◆ TargetLatencyInMs

int Microsoft.Psi.Audio.AudioResamplerConfiguration.TargetLatencyInMs
getset

Gets or sets the target audio latency.

This parameter controls the amount of audio to resample and output at a time, which in turn determines the latency of the audio output. The larger this value, the more audio data is carried in each AudioBuffer and the longer the audio latency. For live audio capture, we normally want this value to be small as possible. By default, this value is set to 20 milliseconds. Is is safe to leave this unchanged.
