Detailed Description

Component that writes messages to a multi-stream store. The store can be backed by a file on disk, can be ephemeral (in-memory) for inter-process communication or can be a network protocol for cross-machine communication. Instances of this component can be created using Store.Create(Pipeline, string, string, bool, Serialization.KnownSerializers).
