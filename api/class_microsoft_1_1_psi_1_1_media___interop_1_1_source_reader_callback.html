Parameters
hrStatusThe status code. If an error occurred while processing the next smple, this parameter contains the error code

llTimestampThe time stamp of the sample, or the time of the stream event indicated in dwStreamFlags. The time is given in 100-nanosecond units
pSampleA pointer to the IMFSample interface of a media sample. This parameter might be NULL.
