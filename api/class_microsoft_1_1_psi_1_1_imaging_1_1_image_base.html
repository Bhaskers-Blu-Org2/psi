Detailed Description

Represents an image, stored in unmanaged memory.

Using this class it is possible as to allocate a new image in unmanaged memory, as to just wrap provided pointer to unmanaged memory, where an image is stored.

◆ ImageBase() [5/5]

Microsoft.Psi.Imaging.ImageBase.ImageBase ( BitmapData  bitmapData,
bool  makeCopy = false 
)

Initializes a new instance of the ImageBase class.

Parameters
bitmapDataLocked bitmap data.
makeCopyIndicates whether a copy is made (default is false).

When the makeCopy parameter is false (default), the image simply wraps the bitmap data. As such, the bitmap data must stay locked for the duration of using the ImageBase object.

If the makeCopy parameter is set to true, a copy of the bitmap data is made, and the bitmap data can be released right after the ImageBase has been constructed.

◆ ToBitmap()

Bitmap Microsoft.Psi.Imaging.ImageBase.ToBitmap ( bool  makeCopy = true)

Creates a Bitmap from the image.

Parameters
makeCopyIndicates whether to make a copy of the image data or not.
Returns
A corresponding Bitmap image.

If the makeCopy parameter is set to true, then the method creates a copy of the image, so the Bitmap stays valid even when the image gets disposed. However, setting this parameter to false creates a Bitmap image which is just a wrapper around the image data. In this case, if the image is disposed, the will no longer be valid and accessing it will generate an exception.
