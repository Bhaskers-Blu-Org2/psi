◆ PixelFormat

enum Microsoft.Psi.Imaging.PixelFormat
strong

Defines the various pixel formats supported by the Image type.

Enumerator
Undefined 

Used when the pixel format isn't defined.

Gray_8bpp 

Defines an grayscale image where each pixel is 8 bits.

Gray_16bpp 

Defines an grayscale image where each pixel is 16 bits.

BGR_24bpp 

Defines an color image format where each red/green/blue component is 8 bits. The byte order in memory is: bb gg rr.

BGRX_32bpp 

Defines an color image format where each red/green/blue component is 8 bits. The byte order in memory is: bb gg rr xx.

BGRA_32bpp 

Defines an color image format where each red/green/blue/alpha component is 8 bits. The byte order in memory is: bb gg rr aa.

RGBA_64bpp 

Defines an color image format where each red/green/blue/alpha component is 16 bits. The byte order in memory is: rrrr gggg bbbb aaaa.

◆ Threshold

enum Microsoft.Psi.Imaging.Threshold
strong

Thresholding modes.

Enumerator
Binary 

Thresholds pixels such that: dst(x,y) = maxvalue if (src(x,y)>threshold) = 0 otherwise

BinaryInv 

Thresholds pixels such that: dst(x,y) = 0 if (src(x,y)>threshold) = maxvalue otherwise

Truncate 

Thresholds pixels such that: dst(x,y) = threshold if (src(x,y)>threshold) = src(x,y) otherwise

ToZero 

Thresholds pixels such that: dst(x,y) = src(x,y) if (src(x,y)>threshold) = 0 otherwise

ToZeroInv 

Thresholds pixels such that: dst(x,y) = 0 if (src(x,y)>threshold) = src(x,y) otherwise
