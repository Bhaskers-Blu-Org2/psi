Detailed Description

Component that aggregates a set of acoustic feature streams computed on an input audio stream into a single component.

The acoustic feature streams available from this component are: LogEnergy, ZeroCrossingRate, FFT, FFTPower, FrequencyDomainEnergy, LowFrequencyEnergy, HighFrequencyEnergy and SpectralEntropy. Use the AcousticFeaturesConfiguration class to control which acoustic features to compute.
