Parameters
startTimeStart time of the read request.
endTimeEnd time of the read request.
tailCountNumber of items to read in a tail read request. Set this to zero for a fixed-range read request.
tailRangeTail range function for a dynamic read request. This function computes the start time of the next read range given the time of the last read item. It is typically used to compute a rolling fixed- duration window over live data. Set this to null for a fixed-range read request.
readIndicesOnlyIndicates whether to read the indices rather than the actual data.
