Detailed Description

Represents the configuration for the SystemSpeechRecognizer component.

Use this class to configure a new instance of the SystemSpeechRecognizer component. Refer to the properties in this class for more information on the various configuration options.

◆ BufferLengthInMs

int Microsoft.Psi.Speech.SystemSpeechRecognizerConfiguration.BufferLengthInMs = 1000
getset

Gets or sets the length of the recognizer's input stream buffer in milliseconds.

Audio arriving on the input stream will be stored in an internal buffer for the speech recognition engine to read as it is able. This buffer will block when full until the recognition engine is able to read from the buffer. Set this value to modify the length of the buffer, which is computed based on the length of audio to buffer in milliseconds and the audio input format. By default, a 1000 ms buffer is used. It is safe to leave this value unchanged.

◆ Grammars

GrammarInfo [] Microsoft.Psi.Speech.SystemSpeechRecognizerConfiguration.Grammars = null
getset

Gets or sets the list of grammar files.

Use this to specify a set of grammar files that the recognizer should use for speech recognition. Grammar files are XML-format files that conform to the W3C Speech Recognition Grammar Specification (SRGS) Version 1.0. If this configuration property is not specified or set to null, the recognizer will use a default context-free grammar used for free text dictation.

Gets or sets the expected input format of the audio stream.

Preferred input audio formats are 1-channel, 16-bit PCM samples. Use the WaveFormat.Create16kHz1Channel16BitPcm or WaveFormat.Create16BitPcm(int, int) static methods to create the appropriate WaveFormat object. If not specified, a default value of 16000 Hz is assumed.

◆ Language

string Microsoft.Psi.Speech.SystemSpeechRecognizerConfiguration.Language = "en-us"
getset

Gets or sets the speech recognition language.

Use this to set the locale for speech recognition. If not specified, this defaults to "en-us" (U.S. English). Other supported locales include "en-gb", "de-de", "es-es", "fr-fr", "ja-jp", "zh-cn" and "zh-tw".
