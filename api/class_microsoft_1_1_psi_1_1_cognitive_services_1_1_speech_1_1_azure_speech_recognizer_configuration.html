Detailed Description

Represents the configuration for the AzureSpeechRecognizer component.

Use this class to configure a new instance of the AzureSpeechRecognizer component. Refer to the properties in this class for more information on the various configuration options.

◆ Language

Gets or sets the speech recognition language.

Use this to set the locale for the speech recognition service. If not specified, this defaults to "en-us" (U.S. English). Other supported locales include "en-gb", "de-de", "es-es", "fr-fr", "it-it" and "zh-cn".

Gets or sets the subscription key.

This component uses the Azure Speech API to perform speech recognition. A Cognitive Services subscription is required in order to use this service. The AzureSpeechRecognizer authenticates with the service using a subscription key associated with the subcription. See https://azure.microsoft.com/en-us/try/cognitive-services/ for more information on obtaining a subscription.
