◆ ThrottleWhenFull

bool Microsoft.Psi.DeliveryPolicy.ThrottleWhenFull
getset

Gets or sets a value indicating whether to block the publisher if the receiver queue is full. The default is to not block, and drop the oldest message instead. Use with care, as it affects all other subscribers to the same publisher and can introduce deadlocks (a blocked publisher cannot process control messages anymore).
