Detailed Description

Represents the configuration for the MicrosoftSpeechRecognizer component.

Use this class to configure a new instance of the MicrosoftSpeechRecognizer component. Refer to the properties in this class for more information on the various configuration options.

◆ BufferLengthInMs

int Microsoft.Psi.MicrosoftSpeech.MicrosoftSpeechRecognizerConfiguration.BufferLengthInMs = 1000
getset

Gets or sets the length of the recognizer's input stream buffer in milliseconds.

Audio arriving on the input stream will be stored in an internal buffer for the speech recognition engine to read as it is able. This buffer will block when full until the recognition engine is able to read from the buffer. Set this value to modify the length of the buffer, which is computed based on the length of audio to buffer in milliseconds and the audio input format. By default, a 1000 ms buffer is used. It is safe to leave this value unchanged.

Gets or sets the list of grammar files.

Use this to specify a set of grammar files that the recognizer should use for speech recognition. Grammar files are XML-format files that conform to the W3C Speech Recognition Grammar Specification (SRGS) Version 1.0. At least one grammar is required in order to use the MicrosoftSpeechRecognizer.

Gets or sets the expected input format of the audio stream.

Preferred input audio formats are 1-channel, 16-bit PCM samples. Use the WaveFormat.Create16kHz1Channel16BitPcm or WaveFormat.Create16BitPcm(int, int) static methods to create the appropriate WaveFormat object. If not specified, a default value of 16000 Hz is assumed.

◆ Language

string Microsoft.Psi.MicrosoftSpeech.MicrosoftSpeechRecognizerConfiguration.Language = "en-us"
getset

Gets or sets the speech recognition language.

Use this to set the language for speech recognition. The language must be an installed Runtime Language for speech recognition. The Microsoft Speech Platform Runtime does not include any Runtime Languages for speech recognition. You must download and install a Runtime Language for each language in which you want to recognize speech. A Runtime Language includes the language model, acoustic model, and other data necessary to provision a speech engine to perform speech recognition in a particular language. For a list of supported Runtime Languages and to download them, see http://go.microsoft.com/fwlink/?LinkID=223569.
