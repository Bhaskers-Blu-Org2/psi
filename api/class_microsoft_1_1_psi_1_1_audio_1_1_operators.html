Detailed Description

Stream operators and extension methods for Microsoft.Psi.Audio.

Stream operators and extension methods for Microsoft.Psi.Audio.Windows.

These are a collection of extension methods defining various audio and acoustic operators on AudioBuffer streams, streams of raw audio bytes, streams of floating-point audio samples as well as audio frequency-domain data streams.

These are a collection of extension methods defining various audio operators on AudioBuffer streams.
