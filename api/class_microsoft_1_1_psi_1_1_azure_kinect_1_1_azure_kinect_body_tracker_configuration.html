◆ TemporalSmoothing

float Microsoft.Psi.AzureKinect.AzureKinectBodyTrackerConfiguration.TemporalSmoothing = 0.5f
getset

Gets or sets the temporal smoothing to use across frames for the body tracker.

Set between 0 (no smoothing) and 1 (full smoothing). Less smoothing will increase the responsiveness of the detected skeletons but will cause more positional and orientational jitters.
