Detailed Description

Component that extracts acoustic features (e.g. LogEnergy, ZeroCrossing, FFT) from an audio stream.

The acoustic feature streams available from this component are: LogEnergy, ZeroCrossingRate, FFT, FFTPower, FrequencyDomainEnergy, LowFrequencyEnergy, HighFrequencyEnergy and SpectralEntropy. Use the AcousticFeaturesExtractorConfiguration class to control which acoustic features to compute.
