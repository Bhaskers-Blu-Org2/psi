Detailed Description

Represents a data provider that can provide adapted data to instant data targets. Data of type TSrc supplied to the provider is converted to data of type TDest by the stream adapter and then routed to all instant data targets.

Template Parameters
TSrcThe type of source data.
TDestThe type of destination data.
