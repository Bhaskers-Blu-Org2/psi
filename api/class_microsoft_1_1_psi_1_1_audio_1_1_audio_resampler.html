Detailed Description

Component that resamples an audio stream into a different format.

This component performs resampling on an audio stream of type AudioBuffer to convert it to an alternative format that may be required for consumption by downstream components. The audio format to convert to may be specified via the AudioResamplerConfiguration.OutputFormat configuration parameter in the form of a WaveFormat or WaveFormatEx value.
Please note: This component uses Media APIs that are available on Windows only.
