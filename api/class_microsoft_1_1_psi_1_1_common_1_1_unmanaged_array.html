Detailed Description

Provides efficient, array-style access to data stored in unmanaged memory. The element type can be any simple value type (a primitive type or a struct with no reference-type fields. The simple value types supported by this class can only contain fields of primitive types or other simple value types. For efficient reading and writing of complex types, see the Serialization namespace.

Template Parameters
TThe element type. Must be a blitable, simple value type (a struct with no reference-type fields).
