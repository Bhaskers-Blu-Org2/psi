Constructs a branch termination policy function instance to terminate after a number of messages have elapsed and the corresponding key was no longer present.

Parameters
countThe number of messages that have to elapse with the key not present for the branch to close.

Function indicating whether and when (originating time) to terminate the given branch.

The closing time for the branch is the time of the count-th message that does not contain the key.
